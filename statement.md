Menstrual Cycle Prediction Tool

>Problem Statement

Many people face difficulties planning their activities due to the uncertainty of menstrual cycle dates. A simple tool is needed to analyze past menstrual cycle data and predict the next period start date to aid in personal planning and menstrual health awareness.

>Scope of the Project

*The project focuses on creating a Python-based application that:

Records start and end dates of menstrual cycles.

Calculates the lengths of cycles.

Predicts the next period start date using the average cycle length.

Provides output via the console.     _
It excludes web or mobile app development, user authentication, or advanced health analytics.

>Target Users

Individuals who want to track and predict their menstrual cycles for better planning.

Beginners learning Python programming through a meaningful health-related project.

Health educators looking for simple tools to demonstrate menstrual cycle tracking.

>High-level Features

Input and storage of menstrual cycle start and end dates.

Calculation of cycle lengths based on recorded data.

Average cycle length computation.

Prediction of the expected start date of the next period.

Console-based display of calculations and predicted results.
